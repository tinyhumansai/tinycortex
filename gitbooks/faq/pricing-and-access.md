# Pricing & Access

### How much does TinyCortex cost?

Current pricing for the managed API:

* **$2.00** per 1M input tokens
* **$6.00** per 1M output tokens

At scale (100 queries/day), roughly $9-30/month vs $60-2,250 for standard RAG or $12,000-90,000 for raw frontier models.

### How do I get an API key?

Closed alpha. Email [founders@tinyhumans.ai](mailto:founders@tinyhumans.ai) with your use case.

### Which SDK should I use?

* **Python**: Backend services, data pipelines
* **TypeScript**: Node.js backends, serverless
* **LangGraph**: Native integration for LangGraph agents
* **OpenClaw**: OpenClaw-compatible agent frameworks

All SDKs use the same API key. See[ Choosing an SDK](/broken/pages/MA3pweqUQ6pKDxtRj2iI).

### Can I run benchmarks on my own data?

Yes. See [Run Your Own](../benchmarks/run-your-own.md).
