# OpenClaw Plugin

{% hint style="warning" %}
The OpenClaw plugin is **coming soon**. This page will be updated once the plugin is released.
{% endhint %}

The OpenClaw plugin adds TinyCortex memory to agents built with the [OpenClaw](https://openclaw.ai) framework. Your agents will be able to ingest, recall, and manage memory as part of their tool-calling workflow — with no changes to your existing agent logic.

## What to Expect

* **Drop-in plugin** register once and your OpenClaw agents gain memory tools automatically
* **Same API** uses the same ingest/read/delete memory operations as the other SDKs
* **Credential isolation** API keys are configured at the plugin level, never exposed to the LLM

Stay tuned or [reach out](mailto:founders@tinyhumans.ai) if you'd like early access.
