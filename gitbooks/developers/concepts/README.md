# Concepts

TinyCortex memory is built around a few core primitives. Read this section first, then use [SDK Functions](../sdk-functions/README.md) for implementation details.

## Start Here

- [Memory Items](memory-items.md): the object you store and retrieve
- [Namespaces](namespaces.md): logical isolation for memory groups
- [Ingest](ingest.md): upserting memory into storage
- [Recall](recall.md): retrieving relevant context for LLM prompts
- [Context](context.md): formatted recall output to inject into prompts
- [Delete](delete.md): explicit cleanup of stored memory
- [Memory Decay](memory-decay.md): time-based retention and pruning
- [Interactions](interactions.md): reinforcement signals that affect ranking/retention
- [Timestamps](timestamps.md): temporal signals used during retrieval and decay
